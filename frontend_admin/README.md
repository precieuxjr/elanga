# frontend_admin

Ce dossier est reserve a l'interface d'administration (validation/invalidation
des signalements, gestion des utilisateurs, etc.). Conformement a la demande,
la priorite a ete mise sur `backend` + `frontend_user` (inscription + tableau
de bord du citoyen). Cette interface sera construite dans une prochaine
etape, avec la meme architecture MVC-services que `frontend_user`.
